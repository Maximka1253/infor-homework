# Магазин

Программа автоматизирует простую продажу и учет товаров в магазине.

## Что умеет программа

- хранит товары, категории, цены и остатки на складе;
- позволяет добавить новый товар;
- позволяет добавить товары в корзину;
- проверяет, что покупатель не покупает больше товара, чем есть на складе;
- создает чек с датой, номером чека, списком товаров и итоговой суммой;
- уменьшает остатки товаров после продажи;
- показывает отчет по проданным товарам и выручке за выбранную дату.

## Файлы проекта

- `shop.py` - главный файл запуска программы;
- `gui.py` - графический интерфейс;
- `database.py` - работа с базой данных SQLite и SQL-запросы;
- `REPORT.md` - отчет со схемой базы данных и текстами запросов;
- `store.db` - файл базы данных, создается и обновляется автоматически;
- `receipts/` - папка с текстовыми файлами чеков, создается после продаж.

## Как запустить

Запускать нужно файл `shop.py`:

```bash
python3 shop.py
```

После запуска откроется окно программы.

## Как проверить, что код компилируется

```bash
python3 -m py_compile shop.py gui.py database.py
```

Если команда ничего не вывела, значит синтаксических ошибок нет.

## Важно

Если структура базы данных менялась, старый файл `store.db` может хранить старые таблицы. Для чистого запуска можно удалить `store.db`, тогда программа создаст новую базу при следующем запуске.
